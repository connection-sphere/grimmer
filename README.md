## Grimmer

Grimmer is an operations and growth automation agent that unifies AdsPower, PostgreSQL, and the ConnectionSphere API into a single secure, fast, and scalable workflow.

## Local Environment Installation

For the standard local environment flow, Grimmer installation is automatic through `developers-environment.op` during setup.

Tutorial: https://github.com/massprospecting/.github/blob/main/docs/002-installation-on-local-environment.md
